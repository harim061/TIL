#JSON
> JSON은 JavaScript Object Notation의 약자로, 브라우저와 서버사이에서 오고가는 데이터의 형식


## JSON.stringify
> JSON 문자열로 반환
```
JSON.stringify(value, replacer)
```
- value : JSON 문자열로 변환할 값. 배열, 객체, 등

- replacer: 함수 또는 배열 (기본 값 = 문자열)
