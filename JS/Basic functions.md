## 1-1. Basic functions
```
function sum(x,y){
  return x+y;
}
```

### Returning the result
- in JS you have to return from inside functions
```
function sum(x, y) {
    return x + y;
    console.log("Hello World"); // this will NEVER run
}
```
- return keyword will quit the function with the result, so the code afterward will never run
